## 機能一覧
- ユーザー登録、ログイン機能(device)

- 一般ユーザーの機能
  - 自身の情報確認
    - 登録名やメールアドレスの確認
    - 与えられた就業ルールの確認
      - 出勤する曜日の確認
      - コアタイム・休憩時間上限を確認

  - 勤怠登録
    - 出勤、退勤、休憩時間の打刻機能
    - 押し間違えた打刻の消去機能(5分以内)
    - 押し間違えた打刻の修正申請機能

  - 振替出勤/休暇申請
    - 振替出勤、有給休暇、病欠を理由とした例外日の申請機能

  - 通知機能
    - 各申請の承認/却下確認のための通知機能

- マネージャー権限を持つユーザーの管理機能
  - ユーザーの情報確認
    - ユーザーの登録名やメールアドレスの確認
  
  - メンバー管理(ユーザーをメンバーとして登録可能)
    - メンバーの打刻修正申請の承認/却下機能
    - メンバーの振替/休暇申請の承認/却下機能
    - メンバーの就業規則の更新機能
